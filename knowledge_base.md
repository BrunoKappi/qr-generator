Bkappi QR Code Studio is a highly-engineered, offline-first QR Code customization platform designed for absolute data privacy and premium design outputs. Built on a Single Page Application (SPA) architecture using React 18+, TypeScript, and Vite, the platform uses Tailwind CSS for clean responsive utility styles and Framer Motion (motion/react) to power fluid layout transitions and animations. Its icons are dynamically generated using Lucide React. The application is completely serverless and client-side, running entirely in the browser with 0% telemetry, tracking, or cloud-based data synchronization, relying on standard browser storage (localStorage) for user history and bookmark assets. Supported translation locales are English (en), Português (pt-BR), and Español (es). The application supports a specialized Intelligent Autodetect Input Engine that dynamically parses inputs in real-time to match different QRContentType formats, which include: 'url' (standard website routing), 'text' (general raw text), 'phone' (direct-dial international phone protocols), 'email' (mailto links supporting preset recipient address, subject, and body text), 'wifi' (wireless settings including SSID, password, and security types like WEP, WPA, or None), and 'sms' (prefilled destination number and message body). A built-in Physical Capacity Validation Guard assesses text length and alerts the user to visual density hazards. It leverages four configurable Error Correction Levels (ECC) defined by standard specifications: L (~7% data recovery), M (~15% data recovery), Q (~25% data recovery), and H (~30% data recovery). This ECC system works in tandem with a visual capacity progress bar to notify users if visual details could compromise physical print readability. The Elite Customization & Brand Styling Suite lets users style three independent matrix segments: 1) Dots/Matrix Styles (QRDotsType) supporting 'square', 'rounded', 'dots', 'classy', 'classy-rounded', and 'extra-rounded'; 2) Corners/Eye Frames (QREyeFrameType) supporting 'square', 'dot', 'rounded', 'extra-rounded', 'outpoint', and 'classy'; 3) Corners Pupil/Eye Balls (QREyeBallType) supporting 'square', 'dot', 'rounded', 'heart', 'diamond', and 'classy'. Color selections feature single solid configurations or dual-gradient maps (both 'linear' and 'radial' types) with customizable gradients (dotsGradientColor1 and dotsGradientColor2) and adjustable angles (dotsGradientRotation). Background parameters allow configuring custom colors, transparency flags (backgroundTransparent), and custom opacity sliders. Logo overlays allow importing Base64 files or URLs, resizing the brand badge from 5% to 40% of the total QR grid size (logoSize: 0.05 to 0.4), editing logo margins, corner radius, opacity, and toggling a background 'Safe Zone' (logoClearBackground and logoBgColor) which structurally removes QR code matrix dots directly behind the logo overlay area to guarantee scanning hardware focus. Frame and label custom styling includes presets such as 'none', 'scan_me', 'open_link', 'website', 'instagram', 'whatsapp', 'minimal', 'corporate', 'tech', and 'creative' alongside editable frameText, frameColor, and frameTextColor variables. Image effects support shadow toggling, border thickness, border radius, and rotations (0, 90, 180, and 270 degrees). Exporter formats support vector options (.svg, vector .pdf templates) and raster formats (.png, .jpg) rendered through an isolated workspace using high-definition canvas scalers up to resolutions of 512x512, 1024x1024, 2048x2048, and 4096x4096px (4K print-grade quality). The history dashboard manages previous items using the QRHistoryItem interface to store name, config details, created timestamp, usage counters, and a favorite flag (isFavorite) inside local client databases. Pre-configured presets include Vercel Dark, Classic Slate, Stripe Indigo, Neon Cyber, Sunset Glow, Emerald Green, Royal Gold, and Midnight Purple. Furthermore, the repository hosts a standalone Landing Page inside the `/landing-page` folder written in Vanilla HTML5, Vanilla CSS, and Vanilla JavaScript, showcasing all these key features dynamically. The landing page has an English-first default configuration, a localstorage-persisted light/dark theme switcher using `data-theme` selectors, Google Fonts 'Outfit' integration, custom Lucide icons, responsive flex/grid layouts, scroll-reveal fade animations utilizing Intersection Observer, and a dedicated developer card showcasing systems analyst Bruno Kappi with custom links to his portfolio, LinkedIn, and blog.
