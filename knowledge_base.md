Bkappi QR Code Studio is a professional, high-definition, and highly customizable offline-first QR Code generation platform. Architecture and Tech Stack: Built using React 18+ with TypeScript and Vite, utility-first styling with Tailwind CSS, dynamic physics and animations with Framer Motion (motion/react), and icons from Lucide React. It operates 100% offline with absolute data privacy, no telemetry or cloud-based database syncs. Functionalities: 1. Intelligent Autodetect Input Engine that synthetically parses inputs on-the-fly for URLs, Wi-Fi Networks (SSID, Password, WPA/WEP/None), Emails, Phone numbers, SMS Messages, and Plain Text. 2. Physical Capacity Validation Guard featuring configurable Error Correction Levels (L ~ 7%, M ~ 15%, Q ~ 25%, H ~ 30%), a real-time visual capacity percentage bar to warn about density hazards, and physical safety alerts for older phone cameras. 3. Elite Customization & Brand Styling Suite with options for dots and matrix shapes (Standard, Rounded, Classy, Extra Rounded, Outpoints, Classy-Inverted), independent eye configuration (Eye Frames and Eye Balls), vibrant solid or dual-gradient colors (Horizontal/Vertical), and brand logo overlays (SVG, PNG, JPEG) with customizable size, rotation, and an optional Safe Zone toggler. 4. Multi-Format High Definition Exporting with an isolated export workspace supporting up to 4096 x 4096 px renders, capable of vector downloads (SVG, PDF) and raster downloads (PNG, JPG), as well as universal clipboard copying. 5. Past History & Favorite Templates featuring curated galleries (Classic Slate, Stripe Indigo, Neon Cyber, Vercel Dark, Sunset Glow, Emerald Green, Royal Gold, Midnight Purple), a persistent local database for history, and a favorites ledger for recurring generations. 6. Tri-lingual Internationalization with complete localized translations in English (en), Português (pt-BR), and Español (es). Additionally, the project features a premium static landing page inside the /landing-page directory built with vanilla HTML, CSS (using variables for light/dark themes, glassmorphism, and responsive grids), and JavaScript for dynamic theme toggling, scroll animations using Intersection Observer, and instant language switching between English, Portuguese, and Spanish, alongside a developer showcase.
